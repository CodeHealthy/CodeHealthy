## 🚀 Featured Project

### [SkillSync — Automated Talent Skill Validator](https://skill-sync-frontend-mauve.vercel.app/)

A multi-tenant assessment platform that helps organizations evaluate candidates through quizzes and coding challenges.

- Recruiters can register organizations, invite candidates, create assessments, assign tests, and review results
- Candidates can run code, submit answers, and review scores, feedback, output, and errors
- Secure Docker-based code execution supports **Java, JavaScript, and Python**
- Includes JWT authentication, Google OAuth, email verification, password reset, role-based access control, and organization-level data isolation
- Uses AI to help administrators draft quiz and coding challenges before review and publishing

**Built with:** Java 17 · Spring Boot · React · MongoDB Atlas · Docker · AWS EC2 · API Gateway · Vercel
